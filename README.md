# Personal Website – Portfolio

A personal portfolio website built with React, Vite, and Tailwind CSS.

🌐 **Live site:** https://personal-website-shira.netlify.app/

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- EmailJS (contact form)

## Pages

- **Home** – Hero section
- **About** – Personal info
- **Experience** – Work history
- **Projects** – Project gallery with detail pages
- **Testimonials** – Recommendations
- **Contact** – Contact form via EmailJS

## Features

- Responsive design
- Image zoom modal
- ChatBot component
- Project detail pages

## Getting Started

```bash
cd portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```
